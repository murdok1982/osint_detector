> AI-powered OSINT tool for monitoring and analyzing Telegram channels. Detect threats, extremism, and suspicious activities using GPT intelligence.

## ✨ Features

- 🤖 **AI Analysis**: GPT-powered content analysis
- 📊 **Threat Detection**: Identify suspicious patterns
- 🔍 **Channel Monitoring**: Real-time Telegram monitoring
- ⚠️ **Alert System**: Automated threat notifications
- 📈 **Sentiment Analysis**: Analyze channel sentiment
- 🌐 **Multi-language**: Support for multiple languages
- 💾 **Database**: Store and query historical data

## 🚀 Installation

```bash
git clone https://github.com/murdok1982/osint_detector.git
cd osint_detector
pip install -r requirements.txt
cp .env.example .env
# Configure Telegram API & OpenAI keys
python osint_monitor.py
```

## ⚙️ Configuration

```env
TELEGRAM_API_ID=your_api_id
TELEGRAM_API_HASH=your_api_hash
TELEGRAM_PHONE=your_phone_number
OPENAI_API_KEY=your_openai_key
ALERT_WEBHOOK=your_webhook_url
THREAT_THRESHOLD=medium
```

## 💻 Usage

```python
from osint_detector import ChannelMonitor

monitor = ChannelMonitor()
monitor.add_channel("@suspicious_channel")
monitor.start_monitoring()
```
